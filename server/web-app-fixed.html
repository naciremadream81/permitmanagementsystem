About This Application

Florida Permit Management System

This is a live web application connected to the Permit Management System API.

Features:

  • Real-time connection to the backend API
  • Dynamic county data loading
  • API endpoint testing
  • Responsive design
  • Error handling and user feedback

Technical Stack:

  • Backend: Kotlin + Ktor framework
  • Database: PostgreSQL + Redis
  • Frontend: HTML5 + JavaScript
  • API: RESTful JSON API
  • Deployment: Docker + Nginx

API Endpoints:

  • GET /counties - List all counties
  • GET /counties/{id}/checklist - Get county checklist
  • POST /auth/register - Register new user
  • POST /auth/login - User login
